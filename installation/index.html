Installation Guide¶

Prerequisites¶

Make sure you have Python 3.6 or higher installed on your machine. You can download Python from the official Python website.

Installing BeginnerNN¶

To install BeginnerNN, simply run the following command in your terminal: ```bash pip install BeginnerNN

This will download and install the latest version of BeginnerNN. Verifying the Installation

To verify that BeginnerNN is installed correctly, you can run the following Python script:

import BeginnerNN

print("BeginnerNN is successfully installed!")

If you don't encounter any errors, you're all set to start using BeginnerNN!
